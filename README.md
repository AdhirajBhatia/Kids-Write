# Kids Write

## Kids Write is a blogging platform for kids , where kids can express themselves and share their knowledge and thoughts with the world .

## Kids Write Is Open Source And Has MIT License .

### Technologies Used :

- Node.js
- Express
- Pug
- MongoDB
- Mongoose
- JQuery
- Ajax
- Bootstrap
